新开Minecraft整合包服务器

发表于2025-07-11|更新于2025-07-17|Minecraft Server
|总字数:503|阅读时长:1分钟|浏览量:

欢迎来到我的Minecraft服务器介绍文章，接下来我会详细介绍这个服务器，想要模组包或者服务器IP与端口号，请点击跳转
提示：服务器基本上不会崩溃，但如果崩溃等5分钟自动重启就可以加入了（如果还进不去就再等等）

基本信息

性能

CPU

本服务器的CPU是Intel(R) Core(TM) m3-8100Y CPU

内存

本服务器的内存是4G的，开一个60个模组的整合包够了

网络

本服务器的网络不算太好，可能有轻微回弹，但不影响正常游玩

人数

最多人数

本服务器最多可容纳玩家10人，但满10人可能会出现崩溃等状况，所以尽量不要达到10人

适合人数

本服务器适合人数是1-5人，最佳人数是1-3人

整合包

本服务器的整合包是Create Live 6，但多加了几个模组，所以我把包放在了后边，可以点击跳转

内容介绍

本服务器是纯生存服务器，里面没有任何创造元素，并且任何玩家不允许使用其他方法开启创造模式
由于服务器性能原因，不得使用大型轰炸机或盾构机，也不得进行TNT炸服
本服务器不会给予任何玩家op权限，也不会随意踢出玩家
如果发现阻止服务器正常运行的行为，则会踢出玩家
本服务器允许玩家飞行，所以无需担心使用喷气背包被服务器踢出
由于本服务器用的是内网穿透系统，所以不得攻击服务器IP地址

加入方式

本服务器的IP与端口号是103.40.14.14:45372，注意英文冒号
本服务器的包可以点击下载

提示

由于服务器性能不是太好，所以不要运行大型机器
